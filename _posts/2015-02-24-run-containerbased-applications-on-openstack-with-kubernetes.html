Many developers containerize their application so that it can run on any infrastructure; however, it’s still too hard to run containers on a private cloud.Together with Mirantis, we’ve integrated Kubernetes, our open source container manager, in OpenStack.This integration will make it easier to run your apps on a private cloud, while enabling new “hybrid” cloud possibilities.To learn more, sign up for the waiting list.If your company is bigger than a startup, you probably have both on-premises and public cloud infrastructure to host your portfolio of apps.
